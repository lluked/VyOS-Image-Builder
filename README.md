# VyOS Image Builder

Build a VyOS image from source.

## Why?
 - VM Hypervisor packages are not included in published iso, iso needs to be built from source
 - Reproducable build environment
 - No need for a physical build machine

## [Terraform Proxmox](/terraform-proxmox/README.md)
Create a VyOS iso using Terraform and Proxmox.

## [Vagrant Virtualbox](/vagrant-virtualbox/README.md)
Create a VyOS iso using Vagrant and Virtualbox.
